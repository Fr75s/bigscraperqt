bigscraperqt

Scrape Games with LaunchBox

LaunchBox, now on Linux

Bigscraper-qt is a new tool available for Linux that enables you to scrape game metadata with the LaunchBox Games Database. With it, you are able to gather metadata from the vast library of resources LaunchBox has to offer. From old classics to modern games, you can get metadata with this tool.

Remade Twice for the Best Possible

Bigscraper-qt was remade twice to provide you the best experience possible. Going from a python package to electron app to a Python/Qt app, each redo has made this tool easier and lighter (well, maybe not from python to electron.) Now, in this form, bigscraper-qt has a somewhat native interface.

Simplicity is Key

Bigscraper-qt is a simple, easy to use tool, from installation to exporting. With availability both as a flatpak and AppImage, bigscraper-qt is stupid easy to install. And with the guide, you can scrape without any worry or hassle.

Linux + Pegasus Frontend

Bigscraper-qt was made to be able to easily scrape from LaunchBox on unsupported platforms. It focuses on Pegasus Frontend, a highly customizable tool that provides some incredible themes. Bigscraper-qt makes the hardest part of getting started with Pegasus, that is, getting metadata, easy.
